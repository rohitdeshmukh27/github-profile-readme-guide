## 📊 Stats

Resources for displaying your GitHub activity, contributions, and coding statistics on your profile.

- **[GitHub Readme Stats](https://github.com/anuraghazra/github-readme-stats)** — Dynamic GitHub stats cards showing stars, commits, PRs, issues, and contribution rank. The most popular stats tool with 70k+ stars.

- **[GitHub Streak Stats](https://streak-stats.demolab.com)** — Track and display your daily contribution streak, longest streak, and total contributions. Motivates consistent coding.

- **[Nirzak Streak Stats](https://nirzak-streak-stats.vercel.app)** — An alternative fork of Streak Stats with additional themes and reliability improvements.

- **[GitHub Profile Summary Cards](https://github-profile-summary-cards.vercel.app)** — Generate multiple summary cards including stats overview, language distribution by repos/commits, productive hours, and profile details.

- **[GitHub Metrics](https://github.com/lowlighter/metrics)** — The most powerful and customizable GitHub profile analytics tool. Supports 30+ plugins including languages, achievements, activity, habits, stars, and more. Requires GitHub Actions.

- **[GitHub Profile Trophy](https://github-profile-trophy.vercel.app)** — Display GitHub achievements as trophies categorized by stars, commits, PRs, issues, followers, and repositories.

## 📈 Graphs

Resources for creating visual representations of your GitHub contribution activity.

- **[GitHub Readme Activity Graph](https://github.com/Ashutosh00710/github-readme-activity-graph)** — Beautiful line chart showing your contribution activity over time. Supports multiple themes and customization options. No GitHub Actions required.

- **[Snake Contribution Graph (Platane/snk)](https://github.com/Platane/snk)** — Transform your GitHub contribution grid into an animated snake game. The snake eats your contributions as it moves across the grid. Requires GitHub Actions.

- **[Pacman Contribution Graph](https://github.com)** — Convert your contribution grid into a Pacman animation with dark and light mode support. A fun alternative to the Snake graph. Requires GitHub Actions.

- **[GitHub Contribution Chart Generator](https://github.com/sallar/github-contributions-chart)** — Generate an image of your GitHub contribution chart that can be shared anywhere.

- **[GitHub Unwrapped](https://githubunwrapped.com)** — Get your personalized GitHub year in review with animated statistics and insights (seasonal).

## 🎥 GIFs

Resources for adding animated GIFs and visual elements to your GitHub profile.

- **[Cool GIFs For GitHub](https://github.com/Anmol-Baranwal/Cool-GIFs-For-GitHub)** — The largest curated collection of GIFs specifically for GitHub profiles. Categories include programming, cyber security, React, JavaScript, work environment, coding animations, and tech banners.

- **[Animated Fluent Emoji](https://animated-fluent-emoji.vercel.app)** — Microsoft's Fluent Emoji set in animated GIF format. Includes fire, rocket, laptop, robot, and hundreds more.

- **[Readme Typing SVG](https://readme-typing-svg.demolab.com)** — Create customizable typing animations with multiple lines, fonts, colors, and speeds. Perfect for profile taglines.

- **[GitHub Profile Header Generator](https://leviarista.github.io/github-profile-header-generator/)** — Generate custom animated headers for your GitHub profile with text, colors, and patterns.

- **[GIPHY](https://giphy.com)** — General-purpose GIF search engine. Use developer-related search terms to find tech GIFs.

## 🏷 Badges

Resources for creating and customizing badges for your GitHub profile and repositories.

- **[Shields.io](https://shields.io)** — The industry standard for creating custom badges. Supports any label, message, color, and logo. Used by millions of repositories.

- **[Simple Icons](https://simpleicons.org)** — A collection of 3000+ SVG icons for popular brands. These are the logos used by Shields.io badges.

- **[Badge Generator](https://badgen.net)** — Fast badge generating service. Alternative to Shields.io with a simpler API.

- **[For The Badge](https://forthebadge.com)** — Large, bold badge designs for a more playful look. Great for adding personality to your profile.

- **[Custom Icon Badges](https://github.com/DenverCoder1/custom-icon-badges)** — Create Shields.io badges with custom SVG icons that aren't available in Simple Icons.

- **[Visitor Badge](https://komarev.com/ghpvc/)** — Simple profile visit counter badge. Shows how many people have viewed your GitHub profile.

## ⚡ Icons

Resources for displaying technology icons and skill sets on your profile.

- **[Skill Icons](https://skillicons.dev)** — Generate beautiful, consistent icon sets for your tech stack in a single image. Supports dark and light themes with 100+ technology icons. **Recommended.**

- **[DevIcon](https://devicon.dev)** — A comprehensive icon set for programming languages, frameworks, design tools, and cloud services. 200+ icons available as SVGs.

- **[Tech Stack Icons](https://github.com/marwin1991/profile-technology-icons)** — A curated list of technology icons specifically designed for GitHub profiles with markdown snippets.

- **[Simple Icons](https://simpleicons.org)** — 3000+ SVG icons for popular brands and technologies. Can be used standalone or with Shields.io.

- **[Devicons](https://vorillaz.github.io/devicons/)** — An older but comprehensive icon font for developers and designers.

## 🧩 Widgets

Resources for adding interactive widgets and dynamic content to your profile.

- **[Spotify GitHub Profile](https://github.com/kittinan/spotify-github-profile)** — Display the song you're currently listening to on Spotify. Supports multiple themes and cover art display.

- **[Lanyard / Discord Presence](https://lanyard.cnrad.dev)** — Show your real-time Discord status, activity, and what you're currently playing or listening to.

- **[WakaTime](https://wakatime.com)** — Track your coding activity automatically and display daily/weekly coding hours, languages used, and productivity stats.

- **[Blog Post Workflow](https://github.com/gautamkrishnar/blog-post-workflow)** — Automatically pull your latest blog posts from Medium, Dev.to, RSS feeds, or any source and display them on your profile.

- **[GitHub Readme YouTube Cards](https://github.com/DenverCoder1/github-readme-youtube-cards)** — Automatically display your latest YouTube videos as cards on your GitHub profile.

- **[Holopin](https://holopin.io)** — Collect and display digital badges from hackathons, events, and open source contributions.

- **[Capsule Render](https://capsule-render.vercel.app)** — Generate dynamic header and footer banners with gradients, animations, text, and multiple styles.

## 🔧 Generators

Tools that help you generate a complete GitHub Profile README quickly.

- **[GPRM (GitHub Profile README Maker)](https://gprm.itsvg.in/)** — The best overall generator. Supports social badges, GitHub Stats, Skill Icons, multiple themes, and one-click export with live preview. **Recommended.**

- **[GitHub Profile README Generator](https://rahuldkjain.github.io/gh-profile-readme-generator/)** — A popular generator by Rahul D Jain with a step-by-step wizard. Great for beginners.

- **[ProfileMe](https://www.profileme.dev/)** — A quick and simple profile generator with a modern interface. Good for getting started fast.

- **[README Generator by Arturssmirnovs](https://arturssmirnovs.github.io/github-profile-readme-generator/)** — A minimalist generator focused on creating clean, simple profiles.

- **[Profilinator](https://profilinator.rishav.dev/)** — An advanced generator with extensive customization options and multiple sections.

- **[MyOctocat](https://myoctocat.com/build-your-octocat/)** — Build your own custom GitHub Octocat mascot to use as a profile image or decoration.

## 🔐 Cyber Security

Resources specifically for cybersecurity professionals, ethical hackers, and security researchers.

- **[HackerRank Badge](https://shields.io)** — Create a HackerRank badge using Shields.io to showcase your competitive programming achievements.

- **[Hack The Box Badge](https://www.hackthebox.com)** — Display your Hack The Box rank and achievements (requires API access).

- **[Cool GIFs For GitHub — Cyber Security](https://github.com/Anmol-Baranwal/Cool-GIFs-For-GitHub)** — Animated GIFs specifically themed around cybersecurity, hacking, and security research.

- **[Security Headers](https://securityheaders.com)** — While not a profile tool, you can badge your security score for web projects.

## 💻 Coding Platforms

Resources for displaying your competitive programming and coding challenge stats.

- **[LeetCard (LeetCode Stats)](https://leetcard.jacoblin.cool)** — Display your LeetCode problem-solving stats including easy/medium/hard counts, ranking, and submission heatmap.

- **[HackerRank Badge](https://www.hackerrank.com)** — Showcase your HackerRank certifications and badge levels using Shields.io badges.

- **[CodeChef Badge](https://shields.io)** — Display your CodeChef rating and star level with a custom badge.

## 🌐 Open Source

Resources for highlighting your open source contributions and community involvement.

- **[Holopin](https://holopin.io)** — Earn and display digital badges from Hacktoberfest, hackathons, and open source events.

- **[GitHub Metrics — Achievements Plugin](https://github.com/lowlighter/metrics)** — Use the achievements plugin in GitHub Metrics to highlight your open source milestones.

- **[Open Source Badges](https://shields.io)** — Create custom badges indicating your projects are open source, actively maintained, and welcoming contributions.

- **[All Contributors](https://allcontributors.org)** — Recognize all types of open source contributions (code, design, docs, ideas) with an automated contributors table.

- **[GitHub Sponsors Badge](https://github.com/sponsors)** — If you're a GitHub Sponsor or are being sponsored, display it on your profile.

- **[Awesome Lists](https://github.com/sindresorhus/awesome)** — Reference awesome lists relevant to your expertise areas to establish authority in your domain.
