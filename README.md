## 📊 GitHub Metrics (Advanced)

The most powerful and customizable GitHub profile visualization tool. Generates detailed SVG infographics.

🔗 **Repository:** [github.com/lowlighter/metrics](https://github.com/lowlighter/metrics)
📖 **Guide:** [docs/github-metrics.md](docs/github-metrics.md)

**Features Include:**
- 📊 Languages breakdown
- 🏆 Achievements and milestones
- 📦 Repository statistics
- 📅 Activity calendar
- 👥 Follower analytics
- ⏰ Coding habits
- ⭐ Star history
- 🎭 And 30+ other plugins

**Setup:** Requires a GitHub Actions workflow. See the [full guide](docs/github-metrics.md) for step-by-step instructions.

```yaml
# .github/workflows/metrics.yml
name: Metrics
on:
  schedule: [{cron: "0 0 * * *"}]
  workflow_dispatch:
jobs:
  github-metrics:
    runs-on: ubuntu-latest
    steps:
      - uses: lowlighter/metrics@latest
        with:
          token: ${{ secrets.METRICS_TOKEN }}
          user: torvalds
          template: classic
          config_timezone: Asia/Kolkata
```

> 📝 **Note:** Replace `torvalds` with your GitHub username.

## ❓ Frequently Asked Questions

<details>
<summary><strong>How do I create a GitHub Profile README?</strong></summary>

<br/>

Create a new repository with the **same name as your GitHub username**. For example, if your username is `johndoe`, create a repository called `johndoe`. Add a `README.md` file to this repository, and its content will automatically appear on your GitHub profile page. Make sure the repository is **public**.

</details>

<details>
<summary><strong>How do I add GitHub stats to my profile?</strong></summary>

<br/>

Use [github-readme-stats](https://github.com/anuraghazra/github-readme-stats) by adding this to your README:

```html
<img src="https://github-readme-stats.vercel.app/api?username=torvalds&show_icons=true&theme=tokyonight"/>
```

> 📝 **Note:** Replace `torvalds` with your GitHub username.

See the [full guide](docs/github-stats.md) for more customization options.

</details>

<details>
<summary><strong>How do I add streak stats to my profile?</strong></summary>

<br/>

Use [streak-stats.demolab.com](https://streak-stats.demolab.com) by adding this to your README:

```html
<img src="https://streak-stats.demolab.com/?user=torvalds&theme=highcontrast"/>
```

> 📝 **Note:** Replace `torvalds` with your GitHub username.

This shows your current streak, longest streak, and total contributions. See the [full guide](docs/github-streak-stats.md).

</details>

<details>
<summary><strong>How do I add contribution graphs (Snake / Pacman)?</strong></summary>

<br/>

Both Snake and Pacman contribution graphs require **GitHub Actions** to generate the SVG animations. You need to create a workflow file in your profile repository. See the detailed setup guides:
- [Snake Graph Guide](docs/github-snake-graph.md)
- [Pacman Graph Guide](docs/github-pacman-graph.md)

</details>

<details>
<summary><strong>How do I add animated GIFs to my profile?</strong></summary>

<br/>

Find GIFs from collections like [Cool GIFs For GitHub](https://github.com/Anmol-Baranwal/Cool-GIFs-For-GitHub) and add them using an image tag:

```html
<img src="GIF_URL_HERE" width="400" alt="Description"/>
```

Make sure to set a reasonable `width` to keep your profile looking clean. See the [full guide](docs/github-gifs.md).

</details>

<details>
<summary><strong>How do I create a professional GitHub profile?</strong></summary>

<br/>

Follow these steps:
1. Create a profile README repository (same name as your username)
2. Add a professional hero banner using [Capsule Render](https://capsule-render.vercel.app)
3. Write a concise About Me section
4. Display your tech stack with [Skill Icons](https://skillicons.dev)
5. Add GitHub Stats, Streak Stats, and an Activity Graph
6. Include social links with badges
7. Use the [recommended layout](#-recommended-2026-profile-layout) in this guide

See our [Example Profiles](#-example-profiles) for complete templates.

</details>

<details>
<summary><strong>How do I improve my GitHub profile for job applications?</strong></summary>

<br/>

Focus on these areas:
- **Pin your best repositories** — showcase projects relevant to the role
- **Add a professional README** — first impressions matter
- **Show contribution activity** — use streak stats and activity graphs
- **Display relevant skills** — use Skill Icons for your tech stack
- **Include social proof** — badges, certifications, and achievements
- **Keep it updated** — an active profile signals engagement

</details>
