# 🎯 LeetCode Stats — Competitive Programming Stats for GitHub

> Display your LeetCode problem-solving stats, ranking, and submission heatmap on your GitHub profile.

---

## 📖 Overview

[LeetCard](https://leetcard.jacoblin.cool) generates a beautiful stats card showing your LeetCode activity. It displays problems solved, ranking, and optionally a submission heatmap.

---

## ✨ What It Does

- Shows **problems solved** (Easy / Medium / Hard)
- Displays your **ranking**
- Optionally shows a **submission heatmap**
- Supports **contest stats**
- Multiple **themes** and **fonts**

---

## 🚀 Installation & Setup

Add this to your README:

```html
<img src="https://leetcard.jacoblin.cool/torvalds?theme=dark&font=Karma&ext=heatmap" alt="LeetCode Stats"/>
```

> 📝 **Note:** Replace `torvalds` with your LeetCode username.

## 🎨 Parameters

| Parameter | Description | Options |
|---|---|---|
| `theme` | Color theme | `dark`, `light`, `wtf`, `unicorn`, `nord` |
| `font` | Google Font | `Karma`, `JetBrains+Mono`, `Fira+Code` |
| `ext` | Extensions | `heatmap`, `contest` |
| `border` | Border width | `0`, `1`, `2` |
| `radius` | Border radius | `0`, `10`, `20` |

## ✅ Pros
- Quick setup — just a URL
- Shows comprehensive LeetCode stats
- Heatmap extension is visually appealing
- Multiple themes available

## ❌ Cons
- LeetCode-specific only
- Limited customization compared to other tools
- Depends on external service availability

## 🎯 Best Use Cases
- Competitive programmers
- Job seekers (shows problem-solving skills)
- DSA (Data Structures & Algorithms) learners
