## ЁЯУЦ Overview

[Skill Icons](https://skillicons.dev) by [tandpfun](https://github.com/tandpfun) is the recommended way to display your tech stack on GitHub. It generates a single image containing uniform, professionally designed icons for 100+ technologies.

---

## тЬи What It Does

- Generates a **single image** with all your tech stack icons
- Supports **100+ technologies** and tools
- Offers **dark and light themes**
- Allows **custom icons per row** for layout control
- No setup required тАФ just a URL

---

## ЁЯЪА Installation & Setup

```html
<img src="https://skillicons.dev/icons?i=react,nodejs,python,docker,aws&theme=dark" alt="Tech Stack"/>
```

Replace the icon list with your technologies (comma-separated, no spaces).

---

## ЁЯТ╗ Example Code

### Single Row
```html
<img src="https://skillicons.dev/icons?i=react,nodejs,express,mongodb,nextjs,typescript,python,aws,firebase,docker&theme=dark" alt="Skills"/>
```

### Multiple Rows
```html
<img src="https://skillicons.dev/icons?i=html,css,js,ts,react,nextjs,nodejs,express&perline=8&theme=dark"/>
<br/>
<img src="https://skillicons.dev/icons?i=python,django,flask,mongodb,postgres,redis,docker,aws&perline=8&theme=dark"/>
```

### Light Theme
```html
<img src="https://skillicons.dev/icons?i=react,vue,angular,svelte&theme=light" alt="Frontend"/>
```

## ЁЯОи Parameters

| Parameter | Description | Example |
|---|---|---|
| `i` | Comma-separated icon names | `react,nodejs,python` |
| `theme` | `dark` or `light` | `dark` |
| `perline` | Icons per row | `8`, `10`, `13` |
