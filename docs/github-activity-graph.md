# 📉 GitHub Activity Graph — Visualize Your Contribution Activity

> Display a beautiful line chart of your GitHub contribution activity over time with customizable themes and colors.

---

## 📖 Overview

[GitHub Readme Activity Graph](https://github.com/Ashutosh00710/github-readme-activity-graph) generates a dynamic line chart showing your GitHub contribution pattern. It's one of the most visually appealing ways to display your coding activity.

---

## ✨ What It Does

- Generates a **line chart** of daily contributions over the past year
- Supports **area fill** under the graph line
- Offers **10+ themes** and full color customization
- Updates automatically with your GitHub activity
- No GitHub Actions required — just a URL

---

## 🚀 Installation & Setup

Add this to your README:

```html
<img src="https://github-readme-activity-graph.vercel.app/graph?username=torvalds&theme=dracula&hide_border=true&area=true" alt="Activity Graph"/>
```

> 📝 **Note:** Replace `torvalds` with your GitHub username.

## 🎨 Customization

| Parameter | Description |
|---|---|
| `username` | Your GitHub username |
| `theme` | Color theme |
| `area` | Fill area under the line |
| `hide_border` | Remove border |
| `custom_title` | Custom graph title |
| `bg_color` | Background color |
| `color` | Text color |
| `line` | Line color |
| `point` | Point color |
| `area_color` | Area fill color |

### Popular Themes
`dracula`, `github-compact`, `react-dark`, `rogue`, `merko`, `tokyo-night`, `high-contrast`, `lucid`, `github-light`, `xcode`

## ✅ Pros
- Visually stunning — one of the best-looking profile widgets
- Easy setup — just a URL, no Actions needed
- Full-width display — looks great on any profile
- Customizable themes and colors

## ❌ Cons
- Takes up significant vertical space
- Public instance may hit rate limits
- Limited to contribution data only

## 🎯 Best Use Cases
- Adding a visual centerpiece to your profile
- Showing consistent contribution patterns
- Complementing stats and streak cards
