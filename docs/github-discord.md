# 🎮 Discord Presence — Real-Time Discord Status on GitHub

> Show your Discord status, activity, and what you're playing directly on your GitHub profile using Lanyard.

---

## 📖 Overview

[Lanyard](https://github.com/Phineas/lanyard) exposes your Discord presence via an API, and [lanyard.cnrad.dev](https://lanyard.cnrad.dev) generates a beautiful card showing your Discord status on your GitHub profile.

---

## ✨ What It Does

- Shows your **Discord online status** (online, idle, DND, offline)
- Displays **current activity** (game, music, custom status)
- Shows **Spotify listening activity** if linked
- Updates in **real time**

---

## 🚀 Installation & Setup

### Step 1: Join the Lanyard Discord Server
Join [discord.gg/lanyard](https://discord.gg/lanyard) — your presence is tracked once you're a member.

### Step 2: Get Your Discord User ID
1. Open Discord Settings → Advanced → Enable **Developer Mode**
2. Right-click your username → **Copy User ID**

### Step 3: Add to Your README
```html
<img src="https://lanyard.cnrad.dev/api/YOUR_DISCORD_ID" alt="Discord Presence"/>
```

## ✅ Pros
- Real-time status display
- Shows gaming/music activity
- Clean, professional card design
- Free and open source

## ❌ Cons
- Must join Lanyard Discord server
- Only works while you're in the server
- Privacy concern — exposes your Discord status
- Requires Discord Developer Mode for setup

## 🎯 Best Use Cases
- Gaming-focused profiles
- Community builders
- Showing real-time availability
