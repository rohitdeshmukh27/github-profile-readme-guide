# ЁЯПЖ GitHub Profile Trophy тАФ Achievement Trophies for Your Profile

> Display your GitHub achievements as beautiful trophy cards that showcase your milestones in stars, commits, PRs, issues, followers, and repos.

---

## ЁЯУЦ Overview

[GitHub Profile Trophy](https://github-profile-trophy.vercel.app) by [ryo-ma](https://github.com/ryo-ma) adds gamification to your GitHub profile by displaying achievement trophies. Each trophy represents a milestone in your GitHub activity, ranked from C to SSS+.

---

## тЬи What It Does

- Displays achievement trophies for: **Stars**, **Commits**, **PRs**, **Issues**, **Followers**, **Repositories**
- Ranks trophies: **C тЖТ B тЖТ A тЖТ AA тЖТ AAA тЖТ S тЖТ SS тЖТ SSS тЖТ SECRET**
- Supports **multiple themes** and layout customization
- Updates automatically based on your GitHub activity

## ЁЯОи Customization

| Parameter | Description | Example |
|---|---|---|
| `theme` | Color theme | `tokyonight`, `dracula`, `onedark` |
| `row` | Number of rows | `1`, `2` |
| `column` | Columns per row | `4`, `7` |
| `no-frame` | Remove trophy frames | `true` |
| `no-bg` | Transparent background | `true` |
| `margin-w` | Horizontal margin (px) | `15` |
| `margin-h` | Vertical margin (px) | `15` |
| `rank` | Filter by rank | `SSS,SS,S,AAA,AA,A` |
| `title` | Filter by title | `Stars,Commits` |
