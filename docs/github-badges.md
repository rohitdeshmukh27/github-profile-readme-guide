# 🏷 GitHub Badges — Custom Badges with Shields.io

> Create custom badges for technologies, social links, status indicators, and anything else using Shields.io — the industry standard for README badges.

---

## 📖 Overview

[Shields.io](https://shields.io) is the most widely used badge generation service on GitHub. It creates customizable badges in various styles that can display any text, color, and logo. Used by millions of repositories and profiles worldwide.

---

## ✨ What It Does

- Creates **custom badges** with any label, message, and color
- Supports **3000+ logos** from Simple Icons
- Offers **5 badge styles** (flat, flat-square, plastic, for-the-badge, social)
- Generates **dynamic badges** from APIs (npm downloads, build status, etc.)
- Works as **static badges** for tech stack and social links

---

## 🎨 Badge Styles

| Style | Parameter | Description |
|---|---|---|
| Flat | `style=flat` | Default flat badge |
| Flat Square | `style=flat-square` | Flat with square corners |
| Plastic | `style=plastic` | Gradient with 3D effect |
| For The Badge | `style=for-the-badge` | Large, bold badge |
| Social | `style=social` | GitHub-style social badge |

## ✅ Pros
- Industry standard — used everywhere
- Massive logo library (3000+)
- Multiple styles and full color control
- Both static and dynamic badges
- Free and open source

## ❌ Cons
- URL syntax can be complex for beginners
- Some logos may not be available
- Special characters need URL encoding
